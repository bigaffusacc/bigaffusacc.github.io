Hexo Build Notes

Published on May 14, 2017 in Note with 0 comment

  • Cloud Platform：Heroku

  • Environment：PC (Ubuntu 16.04 LTS)

Curl/Wget , Node.js , Git , Heroku CLI , Hexo

Install

Curl/Wget

$ sudo apt install curl
$ sudo apt install wget

Node.js

NodeSource Node.js Binary Distributions

$ curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs
$ sudo apt install nodejs
$ npm config set registry https://registry.npm.taobao.org
#use taobao npm mirror

Git

$ sudo apt install git

Heroku CLI

$ sudo add-apt-repository "deb https://cli-assets.heroku.com/branches/stable/apt ./"
$ curl -L https://cli-assets.heroku.com/apt/release.key | sudo apt-key add -
$ sudo apt update
$ sudo apt install heroku

or

$ wget -O- https://toolbelt.heroku.com/install-ubuntu.sh | sh

Hexo

$ npm install -g hexo-cli

or

$ sudo npm install -g hexo-cli


Build

Git

$ git config --global user.name "username"
$ git config --global user.email "user@email.com"
$ ssh-keygen -t rsa -C "user@email.com”

Heroku

$ heroku login
$ heroku create <appname>
$ heroku keys:add

Hexo

$ hexo init <folder>
$ cd <folder>
$ npm install
$ hexo server 
#open http://localhost:4000/
$ sudo npm install hexo-deployer-heroku --save
$ gedit _config.yml
#
deploy:
  type: heroku
  repo: <repository url>
$ hexo deploy


End
